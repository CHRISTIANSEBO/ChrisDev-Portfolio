# ChrisDev Portfolio – Frontend Case Study

This project is a **personal portfolio site** designed and developed to highlight my skills in **front-end development, UI/UX design, and responsive layouts**.  
It serves as a practical case study of my ability to turn design concepts into functional, user-friendly websites.

---

## 🎯 Goals
- Create a clean, modern, and responsive portfolio site.  
- Showcase reusable front-end components (header, navigation, cards, grids).  
- Apply **UI/UX best practices** to improve usability and flow.  
- Practice structuring a project repository in a professional way.

---

## 🛠️ Tech Stack
- **HTML5 / CSS3 / JavaScript (ES6)**  
- **Responsive Grid & Flexbox** for layouts  
- **Basic animations/transitions** for interactive feel  
- **Version control with Git/GitHub**

---

## 📂 Project Structure
- `/src` – development files (HTML, CSS, JS, assets).  
- `/design` – initial wireframes, style guide, and design references.  
- `/dist` – production-ready build.

---

## 🚀 Features
- Responsive navigation bar with mobile menu toggle  
- Grid-based portfolio section for projects  
- Contact form (front-end only)  
- Smooth scrolling + basic animations
